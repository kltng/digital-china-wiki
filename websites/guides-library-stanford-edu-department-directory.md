# Department directory

**URL:** https://guides.library.stanford.edu/department-directory

**Discovered via:** LibGuides > stanford

## 摘要

該頁面曾是斯坦福大學圖書館的部門目錄，提供各個圖書館部門的聯繫信息和組織結構。該頁面目前不可用（404錯誤），表明它可能已被移動、重命名或從斯坦福大學圖書館網站上移除。尋找圖書館聯繫信息的用戶應訪問斯坦福大學圖書館主網站以獲取最新的員工目錄和部門聯繫詳細信息。
