# Tag Taxonomy

Controlled vocabulary for `digital-china-wiki`. All `tags:` values must come from this list.

## Resource Type (20 tags)

| Tag | Description |
|-----|-------------|
| `archive` | Archival collections, primary source repositories |
| `bibliography` | Bibliographic databases, citation indexes |
| `catalog` | Library catalogs, search portals |
| `database` | General-purpose research databases |
| `dictionary` | Dictionaries, encyclopedias, reference works |
| `ebook` | E-book platforms, digital publishing |
| `film_video` | Film and video archives |
| `full_text` | Full-text collections, text corpora |
| `gis` | GIS data, spatial analysis |
| `guide` | Research guides, tutorials, how-to resources |
| `map` | Maps, cartographic resources |
| `newspaper` | Newspaper archives, news databases |
| `organization` | Research centers, academic institutions |
| `periodical` | Journals, ejournal platforms |
| `photo` | Photo archives, image collections |
| `platform` | Multi-purpose academic platforms |
| `statistical` | Statistical data, government data, surveys |
| `tool` | Software tools, NLP, OCR, analysis utilities |
| `social_media` | Social media platforms, community sites |
| `website` | General web portals, gateways |
