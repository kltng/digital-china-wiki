# Textref Org

**URL:** https://textref.org/

**Region:** Global

## Summary

TextRef provides a distributed method for sharing metadata about online historical texts and integrating this metadata into searchable catalogs with direct links to electronic resources. This system currently provides access to a catalog of pre-modern Chinese texts available in various online database systems.
