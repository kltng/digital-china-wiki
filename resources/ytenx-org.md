# Ytenx Org

**URL:** https://ytenx.org/

**Region:** Global

## Summary

韻典網是一個綜合韻書查詢工具，包含廣韻、中原音韻、洪武正韻牋、分韻撮要和上古音系。 大宋重修廣韻是公元1008年，由陳彭年等人奉詔，據切韻所修訂成韻書一部。 廣韻共收字二萬六千一百九十四，列三千八百七十四小韻，分爲上平聲、下平聲、上聲、去聲、入聲五卷，共計二百零六韻。 分韻撮要是記載清初粵語語音之韻書，現存最早版本是1782年江湖尺牘分韻撮要合集。 分韻撮要分三十三韻部，每一韻部內小韻以陰平、陽平、陰上、陽上、陰去、陽去、陰入、陽入之序排列，小韻有字義但無反切。 分韻撮要作者及具體成書年代均不詳，但所可確定是此書所記錄語音系統不早於明末清初。
