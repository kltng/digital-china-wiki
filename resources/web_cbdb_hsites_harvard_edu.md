# China Biographical Database Project (CBDB)

**URL:** https://cbdb.hsites.harvard.edu/

**Language:** English

**Region:** China

## Summary

The China Biographical Database is a freely accessible relational database with biographical information about approximately649,533 individuals as of May 2025, currently mainly from the 7th through 19th centuries. With both online and offline versions, the data is meant to be useful for statistical, social network, and spatial analysis as well as serving as a kind of biographical reference.
