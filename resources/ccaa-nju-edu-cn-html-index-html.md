# 中国商业广告档案库 Chinese Commercial Advertisement Archive (1880-1940)

**URL:** https://ccaa.nju.edu.cn/html/index.html

**Region:** PRC

## Summary

English: The Chinese Commercial Advertisement Archive is a bilingual project site hosted by Nanjing University. The homepage presents the archive as part of the Ephemera Project and provides multiple ways to explore materials, including an interactive map interface and a structured search form with fields such as product, company, category, agency, city, year, title, and nationality. The page also exposes subject filters for visual and demographic features, indicating that the archive supports detailed discovery of historical Chinese commercial advertisements.

中文：该页面是南京大学主持的“中国商业广告档案库”项目主页，采用中英双语界面，属于 Ephemera Project 的一部分。首页显示该档案库可通过互动地图和高级检索两种方式进入，检索字段包括商品、公司、分类、代理商、城市、年份、标题、国别等，并提供“实物插图”“情境插图”“男性”“女性”“中国人”“外国人”等主题筛选项，说明其重点在于对中国近代商业广告图像与元数据进行组织、检索和研究利用。
