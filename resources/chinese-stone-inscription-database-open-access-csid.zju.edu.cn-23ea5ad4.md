# Chinese Stone Inscription Database 中國歷代墓誌數據庫 OPEN ACCESS

**URL:** http://csid.zju.edu.cn/tomb

**Language:** Chinese

**Region:** China

## Summary

浙江大學圖書館古籍碑帖研究與保護中心成立於癸巳年，落成於甲午年，揭幕於乙未年，三年磨礪始成，以期厚積薄發。 碑帖中心設有展會合一的大廳、學術研究室及古籍碑帖整理修復室，中心資料室與圖書館金石碑帖特藏合署於中心內。 中心定位為對古籍碑帖整理、出版、鑒定、保護、數字化等多個方面進行綜合研究和實踐的開放性學術機構。 以整理出版圖書館古籍碑帖資源、建設「中國歷代墓誌數據庫」作為主要的學術任務；以不定期舉辦展覽作為促進文化的手段。 碑帖中心將以開放與合作的態勢謀求發展，並努力為弘揚傳統文化作出自己的貢獻。 中心以學術與文化為主要的關注點推進發展。
