# Bass Library

**URL:** https://web.library.yale.edu/building/bass-library

**Region:** Global

## Summary

Find us in the heart of Cross Campus, with comfortable, flexible study spaces, media equipment, and more. Bass Library is connected to Sterling Memorial Library via a tunnel from the Wright Reading Room, below the Sterling Library nave. Natural light flows in from two sunken courtyards.
