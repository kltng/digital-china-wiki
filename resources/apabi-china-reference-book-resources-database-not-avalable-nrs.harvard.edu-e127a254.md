# Apabi China Reference Book Resources Database (Not Available)

**URL:** not_avalable_nrs.harvard.edu

**Discovered via:** LibGuides > yale

## Summary

The Apabi China Reference Book Resources Database was a Chinese digital collection focusing on reference works, dictionaries, encyclopedias, and scholarly reference tools. The platform provided access to authoritative Chinese reference materials essential for research in Chinese studies, including classical texts, historical dictionaries, and specialized reference works across various academic disciplines. While the database was previously available through institutional subscriptions, the current link indicates the service is no longer accessible. The collection served researchers needing Chinese-language reference materials for academic work and language study.

## 摘要

Apabi China Reference Book Resources Database 曾是專注於參考工具書、辭典、百科全書及學術參考資料的中文數位收藏。平台提供中國研究必備的權威中文參考材料，包括經典文本、歷史辭典及各學科專門參考工具書。該資料庫曾透過機構訂閱提供存取，目前連結顯示服務已無法使用。此收藏服務需要中文參考材料進行學術研究與語言學習的研究人員。
